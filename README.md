# 🚀 ResumeForge AI

AI-powered resume analyzer that delivers recruiter-style feedback in seconds.

---

## 🔥 Features

- 📊 Resume Score (0–100)
- 📈 Section-wise Breakdown
- ✅ Strengths Detection
- ⚠ Weakness Identification
- 🚀 Actionable Suggestions

---

## 🧠 Tech Stack

- React.js (Frontend)
- Node.js + Express (Backend)
- Groq API (AI)
- pdf-parse (Resume extraction)

---

## 🎯 What It Does

Upload your resume → Get:
- Score
- Strengths
- Weaknesses
- Suggestions
